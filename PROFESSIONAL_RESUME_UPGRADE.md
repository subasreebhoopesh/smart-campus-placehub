## 🖨️ How to Convert to PDF

After downloading the HTML resume:

1. **Open in Browser:** Double-click the downloaded HTML file
2. **Print:** Press Ctrl+P (Windows) or Cmd+P (Mac)
3. **Save as PDF:** Select "Save as PDF" as the printer
4. **Adjust Settings:**
   - Margins: Default or Minimum
   - Background graphics: ON (to preserve colors)
5. **Save:** Choose location and save
