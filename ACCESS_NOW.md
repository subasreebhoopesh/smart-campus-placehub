## 🎯 Quick Start Guide

### For Admin:
1. Login at http://localhost:8081/admin/login
2. Go to "Companies" → Add companies
3. Go to "Drives" → Create placement drives
4. Go to "HR Credentials" → Create HR accounts
5. Go to "Applications" → Review and respond to students

### For Students:
1. Register at http://localhost:8081/student/register
2. Complete your profile
3. Go to "Opportunities" → Browse drives
4. Click "Apply" on drives you're interested in
5. Go to "Applications" → See HR and Admin responses

### For HR:
1. Login at http://localhost:8081/hr/login
2. Go to "Applications" → Review student applications
3. Click on applications to view details
4. Update status (Shortlist/Select/Reject)
5. Add remarks for students
