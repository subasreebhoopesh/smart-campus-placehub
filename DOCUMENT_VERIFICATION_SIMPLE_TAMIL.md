### Step 1: Student-ஆ Test பண்ணுங்க / Test as Student

**1. Browser-ல open பண்ணுங்க:**
```
http://localhost:8080
```

**2. Student-ஆ login ஆகுங்க:**
```
Email: sreesuba219.2005@gmail.com
Password: student123
```

**3. Left side-ல "My Documents" click பண்ணுங்க**

**4. Document upload பண்ணுங்க:**
- "Upload Document" button click பண்ணுங்க
- Document type select பண்ணுங்க (10th Certificate, 12th Certificate, etc.)
- File choose பண்ணுங்க (PDF or image, max 5MB)
- "Upload" click பண்ணுங்க
- Success message வரும்! ✅

**5. உங்க document பாருங்க:**

### Step 2: Admin-ஆ Test பண்ணுங்க / Test as Admin

**1. Student-லிருந்து logout ஆகுங்க**

**2. Admin-ஆ login ஆகுங்க:**
```
Email: admin@college.edu
Password: admin123
```

**3. Left side-ல "Document Verification" click பண்ணுங்க**

**4. Student documents பாருங்க:**
- எல்லா students-ஓட documents இருக்கும்
- Student name, roll number, branch எல்லாம் காட்டும்
- Document type மற்றும் status காட்டும்

**5. Document verify பண்ணுங்க:**
- "Verify" button click பண்ணுங்க
- Remarks type பண்ணுங்க (optional): "Document verified successfully"
- "Verify" click பண்ணுங்க
- Status "Verified" ஆ மாறும் ✅
- Student-க்கு email போகும் (email configure பண்ணிருந்தா)

**6. அல்லது Reject பண்ணுங்க:**
- "Reject" button click பண்ணுங்க
- Remarks type பண்ணுங்க (required): "Please upload clearer image"
- "Reject" click பண்ணுங்க
- Status "Rejected" ஆ மாறும் ⚠️
- Student-க்கு email போகும்
