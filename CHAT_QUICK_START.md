# 💬 Chat System - Quick Start Guide

## 🚀 Access the Chat

### For Admin:
1. Open: http://localhost:8080
2. Login: `admin@college.edu` / `admin123`
3. Click **"Student Chat"** in left sidebar
4. You'll see list of all students
5. Click any student to start chatting!

### For Students:
1. Open: http://localhost:8080
2. Login: `sreesuba219.2005@gmail.com` / `student123`
3. Click **"Chat with Admin"** in left sidebar
4. Start typing your message!

## 💡 How to Use

### Admin Side:
```
1. See all students in left panel
2. Use search box to find specific student
3. Click student name to open chat
4. Type message in bottom input box
5. Press Enter or click Send button
6. Your messages appear on right (blue)
7. Student messages appear on left (gray)
```

### Student Side:
```
1. Chat opens directly with admin
2. Type your question/message
3. Press Enter or click Send button
4. Your messages appear on right (blue)
5. Admin replies appear on left (gray)
6. Messages auto-refresh every 5 seconds
```
