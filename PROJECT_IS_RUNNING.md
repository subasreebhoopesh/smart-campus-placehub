## 🔍 Troubleshooting

### Registration Not Working?

**Check:**
1. Backend server is running (check terminal)
2. Frontend server is running (check terminal)
3. No errors in browser console (Press F12)
4. MongoDB is connected (check backend terminal)

**Solution:**
- Refresh the page
- Clear browser cache
- Try different browser
- Check QUICK_FIX_REGISTRATION.md

### Can't Login?

**For Students:**
- Use the email and password you registered with
- Make sure you completed registration successfully

**For Admin:**
- Email: admin@college.edu
- Password: admin123

### Page Not Loading?

**Check:**
- Frontend URL: http://localhost:8080
- Backend URL: http://localhost:3001/api/health
- Both should be accessible

- Restart terminals
